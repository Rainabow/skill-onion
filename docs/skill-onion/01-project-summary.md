# The Skill Onion

> 까도 까도 매력이 나오는 인터랙티브 포트폴리오 히어로 섹션 — 양파를 벗길 때마다 디자이너의 스킬셋이 한 겹씩 드러난다.

## 배경 및 목적

- **문제**: 기존 포트폴리오의 스킬 섹션은 정적 텍스트/배지 나열 → 방문자 기억에 잘 남지 않음
- **기회**: 클릭(탭)이라는 능동적 행동 + 명확한 시각 피드백(애니메이션) = 도파민 루프 형성
- **기대 효과**:
  - 방문자가 "직접 양파를 까본" 체험을 통해 스킬 목록을 능동적으로 인지
  - UI/UX 디자이너로서 인터랙션 설계 능력과 프론트엔드 이해도를 작품 자체로 증명
  - 리크루터/클라이언트에게 차별화된 첫인상 각인

## 핵심 기능

| # | 기능 | 설명 | 우선순위 |
|---|------|------|---------|
| 1 | 입체 양파 렌더링 | CSS/SVG 기반 레이어드 그라데이션으로 3D감 있는 양파 비주얼 구현 | 필수 |
| 2 | 껍질 벗기기 인터랙션 | 클릭/탭 시 최외곽 껍질이 "톡! 파스스" 터지며 날아가는 파티클 애니메이션 | 필수 |
| 3 | 스킬 키워드 오픈 | 껍질이 제거될 때마다 해당 레이어의 스킬셋 키워드가 등장하는 전환 애니메이션 | 필수 |
| 4 | 커서 팔로워 | 양파 오브젝트 호버 시 커서가 원형으로 변하고 [ PEEL ] 텍스트가 따라다님 — 클릭 어포던스 직관적 전달 | 필수 |
| 5 | 프로그레스 바 | 양파 옆/하단에 `N / 총겹수 Layers` 인디케이터 또는 게이지 바 표시 — 예측 가능성으로 중도 이탈 방지 | 필수 |
| 6 | 완료 상태 | 모든 껍질을 다 벗기면 최종 메시지 + CTA (포트폴리오 더 보기 / 연락하기) 노출 | 필수 |
| 7 | 사운드 이펙트 | 껍질 제거 시 바스락 효과음 재생 + 우상단 On/Off 토글 버튼 — 기본값: Off | 선택 |
| 8 | 리셋 기능 | 완료 후 또는 언제든 양파를 원래대로 되돌릴 수 있는 리셋 트리거 | 선택 |

## 스킬 레이어 구성안 (예시)

양파 레이어 수: **7~9겹** (스킬 카테고리 수에 맞춰 최종 확정)

| 레이어 (겉→속) | 스킬 카테고리 | 키워드 예시 |
|-------------|------------|-----------|
| 1 (최외곽) | UI/UX Design | Figma · Wireframing · Prototyping |
| 2 | Visual Design | Typography · Color System · Layout |
| 3 | Design System | Component Library · Tokens · Storybook |
| 4 | Frontend | React · MUI · CSS Animation |
| 5 | Motion / Interaction | Framer Motion · Micro-interaction · Transition |
| 6 | Research & Strategy | User Research · Heuristic · Information Architecture |
| 7 | Collaboration | Agile · Design Handoff · Notion · Jira |
| 8 | Tools & Workflow | VSCode · Git · Zeplin · Slack |
| 9 | Soft Skills | Communication · Ownership · Growth Mindset |
| 10 (최내부) | Core Identity | Problem Solver · Detail-Oriented · Perpetually Curious |

> 실제 스킬셋은 구현 전 사용자가 최종 확정

## 대상 사용자

- **주요 사용자**: 리크루터 (채용 담당자) — 포트폴리오 첫 화면에서 강한 인상을 받아야 하는 대상
- **보조 사용자**: 클라이언트 / 협업 파트너 — 역량 범위를 빠르게 파악하고 싶은 대상
- **보조 사용자**: 동료 디자이너 — 인터랙션 레퍼런스 또는 구현 방식에서 영감을 얻고자 하는 대상

## 기술적 범위

- **포함**:
  - 단일 히어로 섹션 (1 페이지 단위 컴포넌트)
  - React 기반 인터랙션 및 상태 관리 (레이어 인덱스)
  - Three.js + @react-three/fiber 기반 3D 양파 렌더링 (볼륨감, 질감, 메시 변형)
  - Framer Motion 기반 껍질 fly-away 타이밍 제어
  - 커스텀 커서 팔로워 (데스크톱 전용)
  - 프로그레스 인디케이터 (게이지 바 또는 텍스트 카운터)
  - 사운드 이펙트 + On/Off 토글 (선택 기능)
  - 반응형 지원 (모바일 탭 / 데스크톱 클릭)
  - Storybook 스토리 문서화
- **언어**: Australian English 전용 (All copy in Australian English)
- **제외**:
  - 백엔드 / 데이터베이스 연동
  - 다국어 지원 (한국어 버전 없음)
  - 포트폴리오 전체 페이지 구현 (히어로 섹션만)
- **제약사항**:
  - 기존 MUI 디자인 시스템 토큰 준수
  - 퍼포먼스: 애니메이션이 과도하게 렌더링 성능을 저하시키지 않아야 함
  - 접근성: 키보드 트리거(Space/Enter) 지원 고려

## 성공 기준

- 방문자가 모든 레이어를 끝까지 벗기는 완료율 > 70% (체감 목표)
- 첫 클릭까지 유도 시간 < 3초 (힌트 없이도 자연스러운 어포던스)
- 모바일/데스크톱 모두 버벅임 없이 애니메이션 재생
